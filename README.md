# Análisis de Incidente: Spear Phishing - Banco Imaginario

Este repositorio contiene el análisis técnico y el informe de respuesta ante incidentes de un ataque de phishing dirigido (Spear Phishing) contra un ejecutivo del **Banco Imaginario**.

##  Resumen del Proyecto
El objetivo de este ejercicio es identificar las tácticas de ingeniería social, analizar indicadores técnicos (IoCs) en el encabezado y cuerpo de un correo electrónico, y evaluar la peligrosidad de una página de aterrizaje maliciosa.

##  Estructura del Repositorio
- `README.md`: Descripción general del proyecto.
- `analisis_paso_a_paso.md`: Documentación detallada del proceso de investigación.
- `informe_de_incidente.md`: Informe técnico profesional para la directiva.

##  Herramientas y Conceptos Utilizados
- **Análisis de Correo:** Inspección de dominios, errores de redacción y suplantación.
- **Análisis de URL:** Identificación de sitios fraudulentos y tácticas de "typosquatting".
- **Ingeniería Social:** Reconocimiento de tácticas de urgencia y autoridad.
- **Respuesta a Incidentes:** Protocolo de cuarentena y mitigación.
